SELECT ... FROM ... - извелечение данных (столбца) из таблицы
(SELECT) DISTINCT ... - фильтр, возвращающие только уникальные значения
(SELECT) TOP .5. ... - выборка из пяти первых строк таблицы
(SELECT ... FROM ...) LIMIT .5. OFFSET .5. - вернёт пять строк начиная с 5 строки. Первое число - кол-во строк для извлечения,
второе - с какой строки начать.
(--), (#), (/* */) - коментарии в SQL

В теории реляционных баз данных говорится, что порядок извлечения данных не имеет значения, если не указан порядок сортировки.

Инструкции SQL состоят из предложений, одни из которых обязательны, другие нет.           

					ФИЛЬТРАЦИЯ 

(SELECT ... FROM ...) ORDER BY ..., (...) - сортировка в алфавитном порядке, предложение ОБЯЗАТЕЛЬНО должно стоять в конце запроса.

""SELECT prod_id, prod_price, prod_name
FROM Products 
ORDER BY 2, 3;"" - сортировка по относительному номеру в SELECT

""SELECT prod_id, prod_price, prod_name
FROM Products 
ORDER BY prod_price DESC, prod_name;"" - DESC фильтрует столбец по убыванию и только тот столбец, после которого оно стоит,
ASC - сортировка по возрастанию.

""SELECT prod_name, prod_price
FROM Products
WHERE prod_price = 3.49;"" - WHERE условие отбора
